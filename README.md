# ChagasNetwork

Estimating the influence of kinship-based social networks on domiciliary vector infestation and child *Trypanosoma cruzi* infection risk.

## Description

This repository contains data and code required to reproduce the analyses presented in:

**Tseng KK, Gaspe MS, Fernandez MP (2026).**  
*Human social networks modify eco-social drivers of Trypanosoma cruzi transmission in rural Argentina.*

The study evaluates how kinship-based social connectivity and residential mobility influence spatial patterns of domiciliary infestation with *Triatoma infestans* and child *T. cruzi* infection risk using network analysis, generalized linear models, and Bayesian spatial models implemented with INLA.

---

## Data confidentiality and spatial jittering

To protect participant confidentiality, household geocoordinates included in this repository have been **spatially jittered**. Random displacement was applied to original household locations to prevent identification of individual households while preserving the overall spatial configuration of the study area.

All spatial analyses reported in the manuscript were conducted using the original, non-jittered coordinates. The jittered coordinates provided here allow users to reproduce analytical workflows, mapping procedures, and model structure, but should not be used for inference on fine-scale spatial relationships.

Access to original georeferenced data may be requested from the corresponding author subject to ethical approval and data-use agreements.

## Getting started

Users can fork or clone this repository to their local machine.

We recommend maintaining the folder structure shown below to ensure that relative file paths in the scripts function correctly.

All analyses were conducted in **R** using a reproducible package environment managed with **renv**.

To restore the computational environment, run:

renv::restore()


---

## Repository structure

### 1. `data/`

Contains all datasets required to reproduce the analyses, particularly SourceData_public.RData.
The data dictionary for SourceData_public.RData can also be found in this folder.




Geospatial data required for host distribution mapping include an external shapefile (`MAMMALS.shp`) that is not distributed with this repository due to file size (>1 GB).

This file can be downloaded from the **IUCN Red List Spatial Database**:

https://www.iucnredlist.org/resources/spatial-data-download

After download, place the shapefile in the working directory prior to running scripts that generate host distribution maps.

---

### 2. `scripts/`

All code for data processing, network construction, statistical modeling, prediction, and figure generation is contained in the following R scripts.

Scripts should be run **sequentially**:

- `1_DataCleaning.R`
- `2_NetworkAnalysis.R`
- `3_GLMModels.R`
- `4_INLAModel.R`
- `5_PredictionModel.R`

---

### 3. `figures/`

All figure outputs generated by the scripts is saved here.

Subdirectories include:

    network/
    inla/
    prediction/
    maps/


These folders contain manuscript figures, supplementary figures, and intermediate visualization outputs.

---

### 4. `tables/`

All table outputs generated by the scripts is saved here.

Subdirectories include:

    network/
    glmm/
    inla/
    prediction/


These folders contain manuscript (supplementary) tables and additional/intermediate analytical results.

---

### 4. `renv/`

Contains files required to reproduce the package environment used for analysis.

## Note on restricted spatial data

Some spatial data files used in the original analysis are **not included in this repository** in order to protect the privacy and confidentiality of participating households.

### Specifically

- High-resolution shapefiles and polygon boundaries used to visualize the study area are not provided because they could facilitate re-identification of household locations.
- Original raster outputs generated during spatial modeling and mapping are also not included, as these files contain sensitive geocoordinate information.

### Instead, this repository provides

- Processed and spatially transformed (**jittered**) household coordinates  
- All network, sociodemographic, ecological, and analytic datasets  
- Full modeling and figure-generation code  

Sections of the scripts that reference unavailable spatial files are retained for transparency but **can be safely ignored** when reproducing the core statistical and network analyses.

These restrictions **do not affect the ability to reproduce the main results**, parameter estimates, model outputs, or non-sensitive figures reported in the study.
